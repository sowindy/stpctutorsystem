ucenter整合：
1, 建议安装UTF-8编码版本的UCenter
2, 在UCenter上创建应用(登录UCenter用户管理中心 -> 应用管理 -> 添加新应用 -> 自定义安装), 需填写的信息包括:
- 应用名称: 您的站点名称
- 应用的主URL: __SITE__
- 应用类型: 其他
- 应用接口文件名称: uc.php
- 是否开启同步登录: 是
- 是否接受通知: 是
填写完毕后点击提交，然后复制“应用的UCenter配置信息”
3, 将第2步中复制的“应用的UCenter配置信息”粘贴在以下文本框文件中，并点击保存
是否开启ucenter if($setting['ucenter_on']==1){echo 'checked="checked"';} />开启 if($setting['ucenter_on']==0){echo 'checked="checked"';}/>不开启
